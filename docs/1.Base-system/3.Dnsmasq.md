# Dnsmasq
## 1.Introdution
- **dnsmasq** là một `DNS-forwader` và `DHCP-server` lightweight.
- Phục vụ cho các `local` machines.
- Nó hỗ trợ static và dynamic DHCP leases.

## 2. Cấu hình 
- Cấu hình được thực hiện trong file `/etc/config/dhcp`.
